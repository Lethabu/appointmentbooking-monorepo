# CI/CD Pipeline Templates

This directory contains GitHub Actions workflows and deployment automation for Cloudflare infrastructure.

## Components

- `github-actions/` - GitHub Actions workflows
- `scripts/` - Deployment and automation scripts
- `environments/` - Environment-specific configurations
- `workflows/` - Workflow templates

## Workflows

- **Build & Test** - Automated testing and validation
- **Deploy Workers** - Worker deployment pipeline
- **Deploy Pages** - Pages deployment pipeline
- **Database Migration** - D1 database migration automation
- **Infrastructure** - Terraform deployment workflows
- **Security** - Security scanning and compliance
- **Monitoring** - Deployment monitoring and alerting

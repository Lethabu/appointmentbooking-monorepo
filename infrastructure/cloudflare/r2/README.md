# R2 Storage and CDN Infrastructure Configuration

This directory contains Terraform templates for Cloudflare R2 storage and CDN deployment.

## Components

- `main.tf` - R2 bucket configuration
- `cdn/` - CDN and edge cache settings
- `policies/` - Access control policies
- `lifecycle/` - Storage lifecycle management
- `backups/` - Backup storage configuration

## Features

- Multi-tenant bucket isolation
- Automatic CDN integration
- Access control policies
- Lifecycle management
- Backup storage
- Performance optimization

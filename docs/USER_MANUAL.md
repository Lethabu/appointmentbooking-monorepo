## Managing Bookings

### Viewing Appointments

1. Click **Appointments** in the sidebar
2. Use filters to find specific bookings:
   - Date range
   - Status (Pending, Confirmed, Completed, Cancelled)
   - Service type
   - Customer name

### Confirming a Booking

1. Find the pending booking
2. Click **Confirm** button
3. Customer receives automatic WhatsApp confirmation
4. Booking status changes to "Confirmed"

### Cancelling a Booking

1. Find the booking
2. Click **Cancel** button
3. Add cancellation reason (optional)
4. Customer receives cancellation notification
5. Booking fee refund processed (if applicable)

### Rescheduling a Booking

1. Find the booking
2. Click **Reschedule** button
3. Select new date and time
4. Customer receives rescheduling confirmation
5. Calendar updates automatically

### Adding Manual Booking

1. Click **+ New Booking** button
2. Fill in customer details:
   - Name
   - Phone number
   - Email (optional)
3. Select service
4. Choose date and time
5. Add notes (optional)
6. Click **Create Booking**

## AI Agent Monitoring

### Nia (Booking Assistant)

**What Nia Does:**
- Handles booking requests via WhatsApp
- Answers service inquiries
- Checks availability
- Confirms appointments
- Sends reminders

**Monitoring Nia:**

1. Go to **AI Agents** > **Nia**
2. View recent conversations
3. Check resolution rate (target: >90%)
4. Review escalations to human

**Handling Escalations:**

When Nia can't help, she escalates to you:

1. Go to **AI Agents** > **Handoffs**
2. View unresolved escalations
3. Click on a handoff to see conversation
4. Contact customer directly
5. Mark as resolved when done

### Conversation History

- View all AI conversations
- Search by customer phone number
- Filter by date range
- Export conversation logs

## Content Management

### Updating Services

1. Go to **Services**
2. Click on a service to edit
3. Update:
   - Name
   - Description
   - Price
   - Duration
   - Image (optional)
4. Click **Save Changes**

### Adding New Service

1. Go to **Services**
2. Click **+ Add Service**
3. Fill in details
4. Upload image (optional)
5. Set price and duration
6. Click **Create Service**

### Business Hours

1. Go to **Settings** > **Business Hours**
2. Set hours for each day
3. Mark closed days (e.g., Sunday)
4. Set public holidays
5. Click **Save**

### Contact Information

1. Go to **Settings** > **Business Info**
2. Update:
   - Phone number
   - Email
   - Address
   - Social media links
3. Click **Save**

## Troubleshooting

### Common Issues

**Problem: Booking not showing in dashboard**
- Solution: Refresh the page (F5)
- Check if correct date filter is applied
- Verify booking status

**Problem: WhatsApp notifications not sending**
- Solution: Check WhatsApp integration status
- Verify customer phone number format (+27...)
- Contact support if issue persists

**Problem: AI agent not responding**
- Solution: Check AI agent status in dashboard
- Verify OpenAI API key is active
- Escalate to human support

**Problem: Payment not processing**
- Solution: Check Paystack integration status
- Verify customer payment details
- Contact support for payment issues
